# workstamp 💸

> 연봉 넣으면 **터미널에서 실시간으로 돈이 오른다.** 출근이 조금은 견딜 만해짐.

```
💸 +167,222원 [████████░░] 퇴근까지 1:33:55
```

- **의존성 0개, 파일 1개.** 파이썬 표준 라이브러리만 씀
- 주말·한국 공휴일(대체공휴일 포함, 2025–2027) 빼고 **근무일 기준** 일급 계산
- 야근은 0원입니다 — 정시에 카운팅이 멈춰요 🫠

## 설치

```bash
# pipx 권장 (파이썬 환경 안 더럽힘)
pipx install workstamp

# 또는 pip
pip install workstamp

# PyPI 배포 전이라면 저장소에서 직접:
git clone <repo-url> && pip install ./WorkStamp/cli
```

파이썬 3.9+ 만 있으면 됩니다.

## 30초 시작

```bash
workstamp init --salary 5000     # ① 연봉 5,000만원 (만원 단위)
workstamp                        # ② 돈 오르는 거 구경
workstamp stamp                  # ③ 퇴근 전에 도장 쾅 🔴
```

## 명령어

| 명령 | 설명 |
|---|---|
| `workstamp` | 실시간 티커 (1초마다 갱신, Ctrl+C 종료) |
| `workstamp init --salary 5000` | 연봉 설정. `--start 10:00 --end 19:00`으로 근무시간 변경 (기본 09:00–18:00) |
| `workstamp stamp` | 오늘 출근 도장. 이번 달 누적도 같이 보여줌 |
| `workstamp cal` | 이번 달 도장 달력 (`●` = 출근한 날) |

## 옵션

| 플래그 | 설명 |
|---|---|
| `--once` | 한 줄만 출력하고 종료 — tmux/starship 연동용 |
| `--secret` | 금액 끝 3자리만 표시 `+•••,222원` — 옆자리에 연봉 들키지 않기 🕶️ |
| `--plain` | 색상 없이 출력 (상태바가 ANSI를 못 먹을 때) |

시크릿 모드여도 끝 3자리는 계속 바뀌기 때문에 "오르는 재미"는 그대로입니다.

## 상태바에 상주시키기

**tmux** — 일하는 내내 화면 구석에서 돈이 오릅니다:

```tmux
# ~/.tmux.conf
set -g status-interval 5
set -g status-right '#(workstamp --once --plain)'
```

**starship** — 프롬프트 찍을 때마다 갱신:

```toml
# ~/.config/starship.toml
[custom.workstamp]
command = "workstamp --once --plain"
when = true
shell = "sh"
```

**zsh 우측 프롬프트**:

```zsh
# ~/.zshrc
RPROMPT='$(workstamp --once --plain 2>/dev/null)'
```

## 어떻게 계산하나요

- 일급 = 연봉 ÷ 그 해의 근무일 수 (주말·공휴일 제외 — 2026년은 247일)
- 근무 중 금액 = 일급 × 근무시간 경과 비율 (초 단위)
- 점심시간 차감 같은 건 안 합니다. 밥 먹는 시간에도 돈은 벌어야죠

## FAQ

**Q. 데이터는 어디 저장되나요?**
`~/.config/workstamp/` 아래 JSON 두 개. 서버 없음, 계정 없음, 연봉이 밖으로 나갈 일 없음.

**Q. 공휴일이 안 맞는데요?**
2025–2027 한국 공휴일을 내장했고 근사치입니다. 임시공휴일은 못 맞춥니다 (그건 대통령도 몰랐음).

**Q. 세전인가요?**
네, 세전입니다. 세후 보고 싶으면… 마음이 아프니까 아직 안 만들었습니다.

### 2번 — 위젯

```
출근도장 앱에 홈 화면 위젯을 추가해줘 (WidgetKit, 위젯 익스텐션 타겟 신규 생성).

핵심 요구사항 (이게 제일 중요):
- 근무 중에 "지금까지 번 돈"이 위젯에서 계속 올라가는 것처럼 보여야 한다.
  퇴근 후 완성 금액을 보여주는 건 부차적이고, 일하는 동안 숫자가 오르는 게 동기부여 포인트.
- WidgetKit은 초 단위 갱신이 안 되는 걸 알고 있음. 구현 전에
  Text(timerInterval:) / ProgressView(timerInterval:) / 촘촘한 타임라인 엔트리 중
  뭐가 "돈이 오르는 느낌"에 가장 가까운지 조사하고 선택 이유를 먼저 설명해줘.

상태별 표시:
- 근무 중: 오르는 금액 + 진행률
- 퇴근 후 (도장 찍음): "오늘 진짜 벌었다 🤑" + 확정 금액
- 도장 안 찍음 / 쉬는 날: 각각 구분되는 카피

앱과 위젯이 데이터를 공유해야 하니 App Group 설정 포함해줘 (시뮬레이터라 서명 이슈 없게).
디자인은 Theme.swift의 키치 팔레트 재사용. 스몰/미디엄 두 사이즈.
빌드해서 시뮬레이터 홈 화면에 위젯 올린 스크린샷으로 확인까지 해줘.
```

```
- 통화: 로케일 따라 원/元/円. 연봉 입력 단위도 나라별 관습에 맞게
  (한국: 만원 / 일본: 万円 / 중국: 万元 또는 월급 기반인지 조사 후 제안)

시뮬레이터 로케일을 zh-Hans, ja로 바꿔 실행한 스크린샷 각각 확인까지 해줘.
```

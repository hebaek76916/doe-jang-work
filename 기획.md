## 구현 완료

- **v1 코어** (iPhone): 연봉 입력 → 근무일 기준 일급 계산(주말·공휴일 제외, 2025–2027 내장), 달력 도장 찍기/떼기, 월·연 누적
- **키치·MZ 리디자인**: Y2K 스티커 + 네오브루탈리즘 (크림 배경, 핫핑크/라임, 검정 테두리+하드섀도)
- **근무 타이머**: 출퇴근 시각 설정, 5단계 상태머신 — 출근 1시간 전 `돈 벌러 가는 중 🏃` → 근무 중 초 단위 실시간 카운팅 → 퇴근 후 2시간 `다 벌었다 수고했다 🍻` → 이후 `무료봉사 중 🫠`. 야근은 정시에 카운팅 멈춤(결정 완료)
- **iOS 홈 위젯** 스몰/미디엄: 분 단위 타임라인 엔트리로 금액 상승 + `Text(timerInterval:)` 카운트다운 + 시스템 자동 진행바. App Group으로 앱↔위젯 공유
- **macOS 메뉴바 앱**: 티커 초당 갱신(`+6,979원`), 팝오버(라이브 카드 + 도장 쾅 + ⚙️설정 + 접이식 전체 달력)
  - ⚠️ 교훈: `MenuBarExtra`는 LSUIElement 조합에서 아이콘이 안 떠서 **NSStatusItem으로 교체**함. macOS 메뉴바는 AppKit이 정답

**플랫폼 현황**: iPhone ✅ / iPad ❌ (family "1"만) / Mac ✅
